### 4. **Registrations Tab** ğŸ“‹

#### What You Can Do:
- View all customer registrations

#### Registration Statuses:
1. **Pending** (Yellow) - New registration, not yet contacted
2. **Contacted** (Blue) - You've reached out to customer
3. **Paid** (Green) - Customer has paid
4. **Completed** (Green) - Order delivered/completed
5. **Cancelled** (Red) - Registration cancelled

#### How to Manage Registrations:
1. Go to **"Registrations"** tab
2. View table with all submissions
3. **Click on any row** to see full details
4. In the detail popup:
   - View all customer information
   - Update status from dropdown
   - Click **"Update Status"** to save
   - Click **"Contact via WhatsApp"** to send message

#### Filtering & Search:
- **Status Filter:** Show only specific status (e.g., only "Pending")
- **Search Box:** Find customers by name, email, or phone
- Both filters work together

#### WhatsApp Integration:
When you click "Contact via WhatsApp":
- Opens WhatsApp with customer's number
- Pre-filled message with customer details
- Ready to send - just add your message

## ğŸ†˜ Troubleshooting

### "Can't see Admin link in menu"
- Make sure you're logged in
- Check if your email is murerwakimberley@gmail.com or marked as admin
- Refresh the page
- Clear browser cache

### "Announcements not showing"
- Backend server must be running
- Check if you're logged in
- Announcements save to browser storage (local)

### "Can't update registration status"
- Backend server must be running
- Check your internet connection
- Make sure you're logged in with admin account

### "Images not uploading"
- Check image file size (shouldn't be too large)
- Use common formats: JPG, PNG
- Make sure you have internet connection
- Try smaller/compressed images

### "Exchange rate not saving"
- Check you entered valid number
- Don't use currency symbols (just number like 18.50)
- Backend server must be running
- Refresh page and try again
